# Facilidex website
Public website hosted at [https://www.facilidex.com](https://www.facilidex.com)

## Run on macOS
Follow [this guide](https://jekyllrb.com/docs/installation/macos/) to install the following dependencies:
* [Homebrew](https://brew.sh)
* [chruby](https://github.com/postmodern/chruby)
* [Jekyll](https://jekyllrb.com)

Then in Terminal run the following:
```
cd PATH_TO_CLONED_REPO
bundle exec jekyll serve
```
